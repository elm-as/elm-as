### 👋 Qui suis-je ?

Je suis le fondateur et lead developer de **ELMAS CORE Laboratory**, un espace de recherche et de développement d'avant-garde où je conçois des systèmes d'exploitation, des IA locales, des environnements cognitifs adaptatifs et des architectures distribuées.

Mon GitHub public n'est que la partie visible de l'iceberg. **La majorité de mon code (> 60 projets d'ingénierie avancée) est maintenue en privé**, par choix stratégique.

### 🌟 Le Projet Ultime : DayanaOS

Mon chef-d'œuvre actuel est **DayanaOS**, un système d'exploitation Linux ultra-léger et modulaire que j'ai bâti de zéro (Kernel 7.0.8 custom, BusyBox, Alpine Rootfs).
- **Vision :** Un OS pré-configuré pour la Data Science et l'IA locale, sans télémétrie ni dépendance cloud.
- **Capacités :** Mode CLI pur (Initramfs) ou GUI (XFCE4 X11).
- **Stack Intégrée :** DuckDB, SQLite, Python 3.12, Gunicorn/Celery/Redis, Notebook Marimo natif, et client IA local (Ollama / llama.cpp statique).
- **Émulation :** Optimisé pour QEMU (WHPX) avec pont réseau asynchrone vers l'hôte.
